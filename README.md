# testeAstrus
 
Crud para teste pratico da Astrus Digital

Como utilizar:

- Criar banco de dados: Executar o arquivo sql `bancoTesteAstrus.sql`

- Inciar react:
`cd /frontend` e `npm start`

- Iniciar servidor laravel:
`cd /backend` e `php artisan serve`

- Login e senha:
  - Usuario: admin
  - Senha: admin
